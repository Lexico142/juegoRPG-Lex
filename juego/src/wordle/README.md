# Wordle

## Descripción
Este es un juego de Wordle. Tienes que adivinar la palabra secreta en 6 intentos o menos. Cada intento debe ser una palabra válida de 5 letras.

## Instrucciones
1. Ejecutar el programa.
2. Adivinar la palabra secreta en el menor número de intentos posible.

## Funcionamiento
- Introduce cualquier palabra de 5 letras para adivinar la palabra secreta.
- Si la letra está en gris, la letra no está en la palabra.
- Si la letra está en amarillo, la letra está en la palabra pero en la posición incorrecta.
- Si la letra está en verde, la letra está en la posición correcta.
