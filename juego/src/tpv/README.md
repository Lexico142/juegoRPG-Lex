# Supermercado
Este es un pequeño programa simple en Java que simula un sistema básico de compras en un supermercado.

## ¿Qué hace?
El programa muestra un menú interactivo donde puedes:

- Ver la lista de productos disponibles con sus precios
- Añadir productos a tu carrito de compras
- Revisar lo que has añadido al carrito
- Finalizar la compra para obtener un ticket con el total
- Salir del programa

## Productos disponibles
El supermercado tiene 4 productos: Pan (1.15€), Huevos (2.54€), Aceite (4.99€) y Harina (0.99€).
